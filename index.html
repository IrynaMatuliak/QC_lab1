UI Controls Showcase

Radiobutton

Radiobutton – is a control element that allows the user to select a single option from a predefined set. Radio buttons are arranged in groups of two or more and displayed on screen as, for example, a list of circular holes that can contain white space (for unselected) or a dot (for selected). The defining feature is that only one radio button within a group can be active at any given time.

Checkbox

Checkbox – is a graphical user interface element that allows the user to make a binary choice, the option can be either selected (on/true) or not selected (off/false). Checkboxes are used when there are lists of options and the user may select any number of choices, including zero, one, or several. Each checkbox is independent of all other checkboxes in the list, so checking one box doesn't uncheck the others.

Text Input

Text input – is a UI control that allows the user to enter text and data. Text input boxes can span single or multiple lines.

Tabs

Tabs – a navigation UI element that allows multiple documents or panels to be contained within a single window. Tabs act as a navigational widget for switching between different sets of content without opening new windows. Each tab represents a separate view or document. Only one tab’s content is visible at a time, but users can quickly switch between tabs.

Button

Button – is an interactive element within a graphical user interface that allows users to initiate an action or command when clicked or tapped. Buttons typically exhibit different visual states (enabled, disabled, hovered, pressed, focused) to provide feedback to the user about their current status and potential for interaction.

Text Label

Text label – is used to display non-editable text in an interface in order to identify, describe, or provide instructions for something. Text labels can be static elements for displaying information, or interactive elements used in web forms to describe input fields for better accessibility.

Link

Link – is digital reference providing direct access to data by a user's clicking or tapping. It allows the user to navigate from one location to another, usually within an app, website, or document. It’s primarily used for jumping to content or triggering a resource.

Tooltip

Tooltip – is an informational UI element represented as a contextual text box or bubble that appears when a user hovers over, focuses on, or long-presses a UI component and provides extra information about a user interface element. Tooltip disappears automatically when the cursor moves away or after a short delay.

Dropdown List

Dropdown list – is a graphical control element that allows the user to select one option from a list that is hidden until activated either by clicking or hovering. It appears as a single-line field or button initially and then expands a list of options after clicking/tapping on it.

Data Grid

Data grid – is a UI component that presents structured data in a tabular format, usually with rows and columns, allowing users to sort, filter, and navigate large datasets. It’s commonly used in applications that handle records, spreadsheets, or dashboards.
